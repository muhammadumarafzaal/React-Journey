# React useEffect Hook

The `useEffect` hook lets you synchronize a component with an external system. This includes side effects like data fetching, timer subscriptions, and direct DOM interactions.

## Concepts & Walking Examples
- **Timer**: Subscribing to intervals and implementing proper cleanup to avoid memory leaks.
- **Users**: Performing asynchronous API data fetching using standard `fetch` within a lifecycle side effect.
- **WindowSizeTracker**: Listening to browser resize events and removing event listeners upon unmount.
- **First**: Exploring mounting and dependency array tracking.
