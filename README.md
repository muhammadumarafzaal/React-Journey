## 📂 Projects and Modules

Here are the 20+ modules and projects currently included in this repository:

### 🔹 1. Core Fundamentals & Concepts
- **ReactJS Installation Tutorial:** Setup and bootstrapping standard configurations.
- **ReactJS Component & JSX Tutorial:** Understanding the virtual DOM, JSX, and functional components.
- **React component2:** Deep dive into atomic design, reusable components, and props.
- **react 1:** Basic raw React implementation to understand the inner workings without build tools.
- **ReactJS Functions, Events, Arrays & Conditional Rendering:** Handling user interactions, mapping arrays to lists, and dynamic UI rendering.

### 🔹 2. Data Flow & Styling
- **Props:** Passing properties and data dynamically to components.
- **ReactJS Props Drilling & State Lifting:** Advanced data passing up and down component trees for sibling communication.
- **ReactJS CSS Styling of Components:** Exploring inline styles, styled-components, and modular CSS.
- **Form Handling:** Controlled inputs, multi-input binding, uncontrolled components using refs, and advanced validations.

### 🔹 3. Hooks Deep Dive
- **ReactJS State Management:** In-depth examples using `useState` (Toggles, Counters, UserProfiles).
- **ReactJS useEffect Hook:** Handling asynchronous side effects, API fetching, and lifecycle cleanup.
- **ReactJS useContext Hook:** Managing global states (Theme, Language, Auth) to avoid prop drilling.
- **ReactJS useReducer Hook:** Managing complex state logic using reducer patterns.
- **ReactJS useCallback Hook:** Optimizing rendering by caching function references.
- **ReactJS useMemo Hook:** Memoizing expensive calculations for extreme performance.
- **ReactJS useLayoutEffect Hook:** Synchronous DOM measurements and mutations before repaints.
- **ReactJS UseEffectEvent Hook:** Exploring experimental hooks for event synchronization.
- **ReactJS Custom Hook:** Building reusable, modular logic hooks for clean architectures.

### 🔹 4. Advanced Patterns & Optimization
- **HOC (Higher-Order Components):** Reusing component logic by wrapping presentational components.
- **ReactJS Higher Order Component:** Extended concepts for dynamic HOC architectures.
- **ReactJS createPortal:** Rendering components (like Modals) outside the main DOM hierarchy.
- **ErrorBoundary:** Catching render errors gracefully and providing a premium diagnostics fallback UI.
- **LazyLoadingReactJs:** Code splitting using `React.lazy` and `Suspense` for faster load times.
- **RouterReactJs:** Implementing complex client-side routing, protected routes, and dynamic parameters.

### 🔹 5. Standalone Applications
- **my-app:** A clean Vite-based application template environment.
- **TODO-List in React:** A robust task management tool to create, complete, and track daily tasks.
- **Expense Tracker:** A financial dashboard tracking incomes and expenses with dynamic UI updates.
- **Shopping Cart Project:** Interactive cart demonstrating state updates and total calculations.
- **E-commerce App:** Comprehensive application with product browsing, detailed views, and cart management.
- **Movie Explorer:** Interactive search app pulling live data using the Fetch API.
- **ReactJS Quiz App Project:** A dynamic trivia application featuring scoring, timer logic, and result summaries.
- **AI Productivity Dashboard:** High-end dark-themed dashboard featuring drag-and-drop tasks, charts, and habits.
- **React CRUD:** Database-driven User Management dashboard built with PrimeReact datatables and Axios API integrations.

## 🚀 Getting Started

To explore any of these projects locally, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/muhammadumarafzaal/React-Journey.git
   ```
2. **Navigate to the specific project directory:**
   ```bash
   cd React-Journey/<project-folder-name>
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Run the development server:**
   ```bash
   npm run dev
   # or
   npm start
   ```
