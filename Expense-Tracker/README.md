# Expense Tracker

A modern React application designed to help users manage their personal finances effectively. Track incomes, expenses, and view your balance instantly.

## Features
- Add and remove expense or income items
- Calculate total balance automatically
- Clean, user-friendly interface
- Local storage support for persistent data (if applicable)

## Technologies Used
- React.js (Hooks: useState, useEffect)
- CSS for styling

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm start` to start the development server
