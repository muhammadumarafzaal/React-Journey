# Shopping Cart Project

An interactive shopping cart application demonstrating core e-commerce functionalities using React. Users can add items to the cart, adjust quantities, and view the total price.

## Features
- Product listing
- Add to cart functionality
- Adjust item quantities (increase/decrease)
- Real-time calculation of total price

## Technologies Used
- React.js
- State Management
- CSS3 for responsive layout

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm start` to start the development server
