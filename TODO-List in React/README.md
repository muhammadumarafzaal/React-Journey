# TODO-List in React

A sleek, functional task management application built using React. This app allows users to create, complete, and manage their daily tasks with ease.

## Features
- Add new tasks
- Mark tasks as complete or incomplete
- Delete tasks
- Interactive and dynamic user interface

## Technologies Used
- React.js
- CSS3

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm start` to start the development server
