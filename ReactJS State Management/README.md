# React State Management

State represents the dynamic data within a React component that can change over time. This project demonstrates core state management capabilities using the `useState` hook.

## Concepts and Components Covered
- **Counter**: Basic increment/decrement state updates.
- **LikeButton**: Simple toggle states.
- **ToggleText**: Showing/hiding UI elements conditionally.
- **UserProfile**: Storing and updating objects in state.
- **Student**: Storing arrays and collection lists in state.
- **InputExample**: Two-way data binding for input elements.
