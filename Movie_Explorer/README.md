# Movie Explorer

A professional React application for browsing and discovering movies. Features include fetching real-time movie data, searching for specific titles, and viewing detailed movie information.

## Features
- Browse popular movies
- Search functionality to find specific titles
- View detailed information including ratings and release dates
- Fully responsive design

## Technologies Used
- React.js
- HTML5 / CSS3
- Fetch API for network requests

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm start` to start the development server
